Un proyecto personal... y familiar

Junto a mi hija - estudiante de último año de Diseño Gráfico -, buscábamos un proyecto común en el que trabajar juntos y compartir conocimientos. De este modo nace, en Septiembre de 2023, darkblobcine.com.

Mientras el diseño y el contenido lo llevamos a cabo juntos, yo soy el encargado de programar todo el site desde 0, apoyándome en mis conocimientos de HTML, CSS y vanilla Javascript.

Una propuesta para los amantes del cine más radical

El site incluye noticias y reseñas de películas de género, muchas de las cuáles están actualmente disponibles en las distintas plataformas de streaming.

Con apenas un año de vida, el blog ha tenido muy buena acogida y ya empezamos a recibir invitaciones, por parte de las distribuidoras, para ver sus películas en formato preestreno y, de esta manera, poder comentarlas o reseñarlas en nuestra página.

Amor por el cine, el diseño y la programación

Darkblobcine.com incluye animaciones y funcionalidades avanzadas que se han programado con un uso intenso de CSS y Javascript:

  • Posibilidad de guardar tus películas favoritas.
  • Un sistema de filtrado de películas por género, año y plataforma de streaming.
  • Un sistema de puntuación y comentarios para las reseñas.
  • Un buscador por keyword o por plataforma de streaming.
  • Integración con la API de JustWatch para mostrar en qué plataforma puede verse actualmente la película.
  • Integración con la API de TMDB (The Movie Database) para extraer el casting de las películas.
