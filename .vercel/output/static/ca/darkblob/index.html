Un projecte personal... i familiar

Al costat de la meva filla - estudiant d'últim any de Disseny Gràfic -, buscàvem un projecte comú on treballar junts i compartir coneixements. D'aquesta manera neix, al setembre de 2023, darkblobcine.com.

Mentre el disseny i el contingut el duem a terme junts, jo sóc l'encarregat de programar tot el site des de 0, recolzant-me en els meus coneixements de HTML, CSS i vanilla Javascript.

Una proposta per als amants del cinema més radical

El site inclou notícies i ressenyes de pel·lícules de gènere, moltes de les quals estan actualment disponibles a les diferents plataformes de streaming.

Amb tot just un any de vida, el bloc ha tingut molt bona acollida i ja comencem a rebre invitacions, per part de les distribuïdores, per veure les seves pel·lícules en format preestrena i, així, poder comentar-les o ressenyar-les a la nostra pàgina.

Amor pel cinema, el disseny i la programació

Darkblobcine.com inclou animacions i funcionalitats avançades que s'han programat amb un ús intens de CSS i Javascript:

  • Possibilitat de guardar les teves pel·lícules preferides.
  • Un sistema de filtratge de pel·lícules per gènere, any i plataforma de streaming.
  • Un sistema de puntuació i comentaris per a les ressenyes.
  • Un cercador per keyword o per plataforma de streaming.
  • Integració amb la API de JustWatch per mostrar en quina plataforma es pot veure actualment la pel·lícula.
  • Integració amb la API de TMDB (The Movie Database) per extreure el càsting de les pel·lícules.
